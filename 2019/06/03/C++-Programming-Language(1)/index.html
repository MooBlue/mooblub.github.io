C程序结构

C Hello World 实例

C程序主要包括以下几个部分：

  1. 预处理指令
  2. 函数
  3. 变量
  4. 语句&表达式
  5. 注释

实例

注释：

  1. 程序的第一行 #include <stdio.h> 是预处理器指令，告诉C编译器在实际编译之前要包含stdio.h文件。
  2. 下一行 int main() 是主函数，程序从这里开始执行。
  3. 下一行 /…/ 将会被编译器忽略，这里放置程序的注释内容。它们被称为程序的注释。
  4. 下一行 printf(…) 是C中另一个可用的函数，会在屏幕上显示消息”Hello, World!”。
  5. 下一行 return 0; 终止 mian() 函数，并返回值0。

编译 & 执行 C 程序
如何把源代码保存在一个文件中，以及如何编译并运行它。下面是简单的步骤：

  1. 打开一个文本编辑器，添加上述代码。
  2. 保存文件为 hello.c。
  3. 打开命令行提示符，进入到保存文件所在的目录。
  4. 键入 gcc hello.c，输入回车，编译代码。
  5. 如果代码中没有错误，命令提示符会跳到下一行，并生成 a.out 可执行文件。
  6. 键入 a.out 来执行程序，屏幕显示 “Hello, World!”。
