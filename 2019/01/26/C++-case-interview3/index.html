数组和指针的区别

数组要么在静态存储区创建，要么在栈上创建。指针可以随时指向任意类型的内存块。

当数组作为函数参数进行传递时，该数组退化成指针；

数组名不能自加自减，但指针可以

int(*s[10])(int)

函数指针数组，S[10] 里面每个元素都是函数指针，指向函数的类型是 int fun(int a )

18. 内存分配方式

堆： 有内存碎片的问题。 一定的算法去找合适的内存。效率低， os 有记录空闲内存地址的链表。
栈： 专门的寄存器存放栈地址。效率高。有大小限制。
自由存储区： 用 malloc / free 分配释放。 和堆类似。
全局/静态存储区： 全局变量，静态变量。
常量存储区：存放常量，不允许修改。

栈内存与文字常量区域

char str1[] = “abc”;
char str2[] = “abc”;
const char str3[] = “abc”;
const char str4[] = “abc”;
const char str5 = “abc”;
const char str6 = “abc”;
char str7 = “abc”;
char str8 = “abc”;
cout << ( str1 == str2 ) << endl;//0 分别指向各自的栈内存
cout << ( str3 == str4 ) << endl;//0 分别指向各自的栈内存
cout << ( str5 == str6 ) << endl;//1指向文字常量区地址相同
cout << ( str7 == str8 ) << endl;//1指向文字常量区地址相同
结果是：0 0 1 1
　　
解答：str1,str2,str3,str4是数组变量，它们有各自的内存空间；而str5,str6,str7,str8是指针，它们指向相同的常量区域。
