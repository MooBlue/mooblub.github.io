1. Hexo 简介

Hexo 是一个基于Node.js的静态博客框架， 使用Markdown解析文章，用户在本地安装Hexo并进行写作，可以方便的生成静态网页托管在github和Heroku上。
Hexo官方网站：https://hexo.io/zh-cn/
